# Полезные утилиты и команды bash:

## Операционная система

*получить версию ОС*

```
lsb_release -a
```

*получить подробную информацию об ОС*

```
cat /etc/lsb-release
```

## Утилиты

*встроенный в bash калькулятор*

```
bc -l
```
*проверка выполнения последней команды или программы*

```
echo $?
```

*управление аудиопотоками*

```
alsamixer
```

*вывод на экран таблицы символов ASCII*

```
ascii
```
2-х *панельный консольный файловый  менеджер*

```
mc
```

*скачать из интернета какой-нибудь файл или сайт на свой компьютер.*

```
wget
```

**например:**

*узнать погоду в твоём городе*

```
wget -O - wttr.in -q
```
*узнать погоду в указанном городе*

```
wget -O - wttr.in/Moscow -q
```
*календарь на 2023 год.*

```
cal 2023
```
*календарь на текущий год*

```
cal
```
*текущие дата и время*

```
date
```

*получить информацию по домену*

```
curl -I https://ya.ru
```
*получить информацию о пользователях ОС*

```
w
```
*получить информацию о группах, в которых состоит текущий пользователь*

```
id
```
*показать все группы в ОС*

```
groups
```
или так:

``` 
less /etc/group 
```

*показать бинарное содержимое файла*

```
xxd -b main.bin
```

*показать текст в бинарном виде*

```
echo 'Hello, World!' | xxd  -b
```

## Управление Терминалом:

*очистка экрана Терминала*

```
Ctrl-L
```

*вывод списка истории введённых пользователем команд*

```
history
```

*очистить историю bash*

```
history -c
```

*вызов из списка команды с указанным номером*

```
!1963
```

*вызов последней выполненной команды*

```
!!
```

*завершение сеанса в Терминале*

```
exit
```

*выбрать Терминал по умолчанию*

```
sudo update-alternatives --config x-terminal-emulator
```

## Управление оборудованием:

*подать звуковой сигнал через встроенный в материнскую плату Buzzer*

```
echo -e \\a
```

*перезагрузить компьютер*

```
reboot
```

*выключение компьютера в указанное время*

```
sudo shutdown -h 00:00
```

*перезагрузить компьютер*

```
shutdown -r now
```

*выключить компьютер*

```
poweroff
```

*выключить компьютер*

```
shutdown -h now
```

*вывод на экран справки по указанной команде*

man <имя_команды>

*например* 

```
man ascii
```

*показать архитектуру процессора*

```
arch
```

*показать  название и версию ОС, версию ядра, текущую дату*

```
uname -a
```

*показать версию ядра Linux*

```
uname -r
```

*показать основную информацию об оборувании*

```
inxi -Fs
```

*показать общую информацию об оборудовании*

```
sudo lshw
```

*показать краткую информацию об оборудовании*

```
sudo sensors-detect
```

*показать информацию о процессоре*

```
cat /proc/cpuinfo
```

*показать прерывания*

```
cat /proc/interrupts
```

*показать объем оперативной памяти в мегабайтах*

```
free -m
```

*показать все подключенные устройства PCI*

```
lspci
```

*показать подробную информацию о всех устройствах PCI*

```
lspci -v
```

*показать информацию о всех подключенных устройствах USB*

```
lsusb
```

*показать сообщения ядра в стандартном потоке вывода*

```
dmesg
```

*показать все запущенные в текущий момент программы*

```
ps -a
```

*показать все запущенные программы и процессы*

```
ps -d
```

*показать все запущенные программы и процессы*

```
ps -ux
```

*показать информацию о видеоконтроллере*

```
lspci -k | grep -E "VGA|3D"
```

*показать информацию о видеодрайвере*

```
lspci -k | grep VGA -A2
```

### Сеть

*настройка сетевых соединений*

```
nmtui
```

*покажет модель вашего WiFi-модуля*

```
lspci -knn | egrep 'Eth|Net'
```

*вывод на экран статус сетевых соединений*

```
ip -c link show
```

*покажет статус сетевых соединений и ip компьютера*

```
ip -c a
```

*показть все сетевые интерфейсы*

```
ifconfig
```

*показать  имя компьютера*

```
hostname
```

*показать IP компьютера*

```
hostname -I
```

*показать активные соединения*

```
sudo netstat -tulpn
```

*показать, какому процессу принадлежат соединия*

```
ss -tp` или `ss -tl
```

*проверка доступности ресурса в интернете или локальной сети 4 раза*

```
ping -c 4 1.1.1.1
```

*проверка доступности ресурса в интернете или локальной сети бесоконечное количество раз*

```
ping
```

### Работа с постоянными носителями информации (HDD/SSD/SD/FLASH etc.)

*показать разметку диска*

```
sudo fdisk -l
```

*вывод информации о вашем жесткои диске*

```
sudo fdisk -l /dev/sd*
```

*показать все устройства хранения информации (жёсткие диски, cd-карты и флешки)*

```
lsblk
```

## Операции с файлами и каталогами:

*показать текущую директорию*

```
pwd
```

`cat` - *просмотр  файла. Например,* `cat filename.txt`

`cd` - *переход  в каталог. Например,* `cd ~/dirname`


`cd ..` - *возврат в предыдущий каталог*


- *переход в домашний каталог пользователя*

```
cd ~
```

*просмотр  содержимого каталога*

```
ls
```

*оказать расширенный список всех файлов (разрешения, владелец, размер и дата изменения)*

```
ls -a
```

*показать список всех файлов, включая скрытые*

```	
ls -l
```

*вместо ls -l можно использовать команду:*

`ll`

*показать расширенный список всех файлов, отсортированный по дате изменения(сначала старые)*

```	
ls -lS
```

*показать расширенный список файлов, с размерами в более читаемом виде(KB, MB, GB)*

```
ls -la
```

*встроенный текстовый  редактор командной строки*

```
nano
```

*устанавливаемый тектовый редактор командной строки (предпочтительно)*

```
micro
```

`mkdir -v` - *создание  нового каталога. Например,* `mkdir newdir`

`cp -i` - *копирование  файла. Например,* `cd -i myfile.txt ~/dirname`

`mv` - *перемещение  или переименование файла или каталога. Например,* `mv filename.txt /Dirname`

`rm -i` - *удаление  файла. Например,* `rm -i filename.txt`

*удаление  директории*

```
rm -ir
```

*показать  объём занятого ОС пространства на диске*

```
df -h
```

*общий  размер текущего или другого (указать какого) каталога в Mb*

```
du -sh
```

*или du -hsx*

*определить  тип файла*

```
file
```

*сделать  указанный файл исполняемым*

```
chmod +x
```

*обновить  базу поиска файлов и каталогов*

```
updatedb
```

*поиск  файлов и каталогов*

```
locate
```

*поиск  файлов и каталогов*

```
find
```

*поиск  исполняемых файлов (программ)*

```
which
```

*показать  местоположение исполняемых файлов*

```
whereis
```

## Утилиты командной строки для развлечения

*показать шахматную доску*

```
printf '%b%b%b%b\n' {,,,}{'\e[47m  \e[m  ','  \e[47m  \e[m'}{,,,}
```

*показать цветную шахматную доску*

```
printf '%b%b%b%b\n' {1..4}'\b'{{1..4}'\b\e[42m  \e[m  ',{1..4}'\b  \e[42m  \e[m'}
```

*показать паровозик*

```
sl
```
